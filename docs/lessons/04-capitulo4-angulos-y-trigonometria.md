# Capítulo 4 — Ángulos y Trigonometría

## Lección: Ángulos y Sistemas de Medición (`angles`)

- **Meta**: convertir grados↔radianes y resolver longitudes de arco.
- **Procedimiento**:
  1. Seleccionar factor de conversión correcto ($180° \leftrightarrow \pi$ rad).
  2. Convertir unidad angular.
  3. Para arco, usar $s=r\theta$ con $\theta$ en radianes.
- **Razonamiento**: el radián relaciona directamente ángulo y arco sobre circunferencia.
- **Errores frecuentes**: usar fórmula de arco con grados sin convertir.

## Lección: Razones Trigonométricas (`trig-ratios`)

- **Meta**: relacionar lados y ángulos en triángulos rectángulos.
- **Procedimiento**:
  1. Identificar respecto al ángulo: cateto opuesto, adyacente, hipotenusa.
  2. Elegir razón (sen, cos, tan).
  3. Plantear ecuación y despejar.
- **Razonamiento**: cada razón es una proporción geométrica estable para un ángulo dado.
- **Errores frecuentes**: intercambiar opuesto y adyacente.

## Lección: Identidades Trigonométricas (`trig-identities`)

- **Meta**: simplificar expresiones y verificar igualdades.
- **Procedimiento**:
  1. Reescribir expresiones con identidades fundamentales.
  2. Simplificar un lado por vez.
  3. Comparar resultados equivalentes.
- **Razonamiento**: las identidades provienen de relaciones invariantes en circunferencia unitaria.
- **Errores frecuentes**: operar ambos lados simultáneamente sin control algebraico.

## Lección: Triángulos Rectángulos (`right-triangles`)

- **Meta**: hallar lados/ángulos faltantes.
- **Procedimiento**:
  1. Determinar datos conocidos (lado-lado, lado-ángulo, etc.).
  2. Elegir Pitágoras o razón trigonométrica.
  3. Resolver incógnita y validar coherencia geométrica.
- **Razonamiento**: Pitágoras y razones trigonométricas describen la misma estructura rectángula.
- **Errores frecuentes**: usar razón trigonométrica no compatible con los datos.

## Lección: Triángulos Oblicuángulos (`oblique-triangles`)

- **Meta**: resolver casos SSS, SAS, ASA/AAS y interpretar ambigüedad SSA.
- **Procedimiento**:
  1. Clasificar el tipo de datos disponibles.
  2. Aplicar ley del coseno o seno según el caso.
  3. Resolver magnitudes faltantes.
  4. Verificar suma angular y viabilidad geométrica.
- **Razonamiento**: las leyes de seno/coseno generalizan las relaciones de triángulos rectángulos.
- **Errores frecuentes**: elegir ley incorrecta para los datos iniciales; no revisar el caso ambiguo.
