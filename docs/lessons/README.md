# Documentación de mejora de lecciones

Este directorio reúne la revisión pedagógica y el plan de mejora para fortalecer todas las lecciones de **Matelearn** con explicaciones completas de:

- cada paso,
- cada procedimiento,
- y el razonamiento detrás de cada resolución.

## Archivos

- `00-revision-y-plan-general.md`: diagnóstico global + plan de mejora transversal.
- `01-capitulo1-conjuntos-numericos-y-geometria.md`: explicaciones completas de todas las lecciones del capítulo 1.
- `02-capitulo2-ecuaciones-y-polinomios.md`: explicaciones completas de todas las lecciones del capítulo 2.
- `03-capitulo3-rectas-conicas-y-sistemas.md`: explicaciones completas de todas las lecciones del capítulo 3.
- `04-capitulo4-angulos-y-trigonometria.md`: explicaciones completas de todas las lecciones del capítulo 4.

## Criterio de “explicación completa” usado en esta documentación

Cada lección documentada incluye:

1. **Meta de aprendizaje** (qué debe lograr el estudiante).
2. **Procedimiento paso a paso** (secuencia operativa para resolver).
3. **Razonamiento matemático** (por qué ese procedimiento funciona).
4. **Verificación del resultado** (cómo comprobar la respuesta).
5. **Errores frecuentes y prevención** (qué confusiones evitar).
